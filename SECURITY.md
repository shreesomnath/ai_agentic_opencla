# Security Policy

## Supported Versions

We actively monitor and patch security vulnerabilities. The table below shows the support status for different versions of **Agentic LCA**:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

We take the security of our Life Cycle Assessment software seriously. If you find a security vulnerability, please do **not** open a public issue. Instead, report it privately to the project maintainers:

* Email: **somnath.luitel416@topper.wku.edu** or **somnathluitel360@gmail.com** (AIR-LAB, Western Kentucky University)

Please include a detailed description of the vulnerability, steps to reproduce, and any potential remediation steps in your email. We will acknowledge receipt of your report within 48 hours and work with you to coordinate a patched release before public disclosure.
